NAME

       unpack.csh

SYNOPSIS

       unpack.csh {seed file}

DESCRIPTION

       A  C-Shell  script  which  "unpacks"  or  extracts SAC binary formatted
       files, SAC Pole Zero instrument response files, and station information
       among other things that rdseed can extract.

REQUIRED PARAMETERS

       seed file name

SEE ALSO

       sacdata2inv(1), sacmerge(1), sacswapbytes(1), rdseed
